=======================================================
  MULTIMODAL CLASSIFICATION AGENT — STEPS TO RUN
=======================================================

PREREQUISITES: Python 3.9+ installed on your machine.

-------------------------------------------------------
STEP 1 — Set up virtual environment
-------------------------------------------------------
cd multimodal-classification-agent

python3 -m venv venv

# Activate it:
source venv/bin/activate          # macOS / Linux
venv\Scripts\activate             # Windows

-------------------------------------------------------
STEP 2 — Install dependencies
-------------------------------------------------------
pip install -r requirements.txt

(Takes 1-2 minutes the first time)

-------------------------------------------------------
STEP 3 — Add your Gemini API key
-------------------------------------------------------
1. Copy the example env file:
   cp .env.example .env

2. Open .env in any text editor and replace the placeholder:
   GEMINI_API_KEY=AIzaSy...your-real-key-here...

   Get a FREE key at: https://aistudio.google.com/apikey
   The key must start with AIzaSy...

   NOTE: ML Model mode works WITHOUT a key.
         Only MCP Server mode needs the key.

-------------------------------------------------------
STEP 4 — Train the ML models (one time only)
-------------------------------------------------------
python ml_models/train_all.py

Expected output:
  Training Sensor Model (GMM + KMeans)
    GMM    train accuracy: 100.0%
    KMeans train accuracy: 100.0%

  Training Network Model (Random Forest)
    Train accuracy: 100.0%

  sensor_test.csv → 32 rows
  network_test.csv → 24 rows

-------------------------------------------------------
STEP 5 — Run the app
-------------------------------------------------------
streamlit run streamlit_app.py

Opens at: http://localhost:8501

IMPORTANT: Always use "streamlit run", NOT "python streamlit_app.py"

-------------------------------------------------------
TROUBLESHOOTING
-------------------------------------------------------
"Please pass a valid API key"
  → Key in .env is wrong. Must start with AIzaSy
  → Get it from: https://aistudio.google.com/apikey

"missing ScriptRunContext" warnings
  → Normal warning when app starts, safe to ignore
  → If app doesn't open: use "streamlit run" not "python"

"429 Too Many Requests"
  → Gemini free-tier rate limit hit
  → Wait 60 seconds and try again

"Model not found" error
  → Run: python ml_models/train_all.py

CSV upload error: "column not found"
  → Use the correct sample file for each section:
     MCP Document:      sample_data/document_data.csv
     MCP Sensor:        sample_data/mcp_sensor_data.csv
     MCP Network:       sample_data/mcp_network_data.csv
     ML Sensor:         sample_data/sensor_data.csv
     ML Network:        sample_data/network_data.csv
     Compare (labeled): test_data/sensor_test.csv
                        test_data/network_test.csv
